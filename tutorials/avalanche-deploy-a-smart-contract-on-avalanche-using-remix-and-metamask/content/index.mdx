# Introduction

![](./assets/avalanche-deploy-a-smart-contract-on-avalanche-using-remix-and-metamask-image-1.png)

Avalanche's Primary Network is a subnet that has three chains: P-Chain, X-Chain, and C-Chain. The C-Chain is an instance of the Ethereum Virtual Machine powered by Avalanche’s Snowman consensus protocol. The [C-Chain RPC](https://docs.figment.io/network-documentation/avalanche/rpc-and-rest-api) can do anything a typical Ethereum client can by using the Ethereum-standard RPC calls. The immediate benefits of using the C-Chain rather than Ethereum are all of the benefits of using Avalanche. These properties could considerably improve the performance of DApps and the user experience.

Today, we will deploy and test a smart contract on Avalanche using Remix and MetaMask.

## Step 1. Setting up MetaMask

Log in to MetaMask -> Click the Network drop-down -> Select Custom RPC

![](./assets/avalanche-deploy-a-smart-contract-on-avalanche-using-remix-and-metamask-image-2.png)

**FUJI Testnet Settings:**

- **Network Name**: Avalanche FUJI C-Chain

- **New RPC URL**:

  `https://avalanche--fuji--rpc.datahub.figment.io/apikey/APIKEYGOESHERE/ext/bc/C/rpc`

- **ChainID**: `0xa869`

- **Symbol**: `C-AVAX`

- **Explorer**: [https://cchain.explorer.avax-test.network](https://cchain.explorer.avax-test.network/)

**Avalanche Mainnet Settings:**

- **Network Name**: Avalanche Mainnet C-Chain

- **New RPC URL**:

  `https://avalanche--mainnet--rpc.datahub.figment.io/apikey/APIKEYGOESHERE/ext/bc/C/rpc`

- **ChainID**: `0xa86a`

- **Symbol**: `C-AVAX`

- **Explorer**: <https://cchain.explorer.avax.network/>

**Local Testnet (AVASH) Settings:**

- **Network Name**: Avalanche Local
- **New RPC URL**:[ ](http://localhost:9650/ext/bc/C/rpc)<http://localhost:9650/ext/bc/C/rpc>

## Step 2. Funding your C-Chain address

# Funding your C-Chain address on Fuji

Navigate to <https://faucet.avax-test.network/> and paste your C-AVAX address. All you need to do is add a “C-” prefix and the faucet will switch from AVAX to C-AVAX.

# Funding your C-Chain address on Mainnet

Funding your C-chain address on mainnet is done through a cross-chain swap from X-chain to C-chain.

# Funding your C-chain address on local testnet

Deploy your own faucet. [Tutorial](https://medium.com/avalabs/the-ava-platform-tools-pt-2-the-ava-faucet-48f28da57146)

# Connect MetaMask and deploy a smart contract

Open [Remix](https://remix.ethereum.org/) -> Select Solidity

![](./assets/avalanche-deploy-a-smart-contract-on-avalanche-using-remix-and-metamask-image-3.png)

Load or create the smart contracts that we want to compile and deploy using Remix file explorer.

For this example, we will deploy an ERC20 contract from [OpenZeppelin](https://openzeppelin.com/contracts).

![](./assets/avalanche-deploy-a-smart-contract-on-avalanche-using-remix-and-metamask-image-4.png)

Navigate to Deploy Tab -> Open the “ENVIRONMENT” drop-down and select Injected Web3 (make sure MetaMask is loaded)

![](./assets/avalanche-deploy-a-smart-contract-on-avalanche-using-remix-and-metamask-image-5.png)

Once we injected the web3-> Go back to the compiler, and compile the selected contract -> Navigate to Deploy Tab

![](./assets/avalanche-deploy-a-smart-contract-on-avalanche-using-remix-and-metamask-image-6.png)

Now, the smart contract is compiled, MetaMask is injected, and we are ready to deploy our ERC20. Click “Deploy.”

![](./assets/avalanche-deploy-a-smart-contract-on-avalanche-using-remix-and-metamask-image-7.png)

Confirm the transaction on the MetaMask pop up.

![](./assets/avalanche-deploy-a-smart-contract-on-avalanche-using-remix-and-metamask-image-8.png)

Our contract is successfully deployed!

![](./assets/avalanche-deploy-a-smart-contract-on-avalanche-using-remix-and-metamask-image-9.png)

Now, we can expand it by selecting it from the “Deployed Contracts” tab and test it out.

![](./assets/avalanche-deploy-a-smart-contract-on-avalanche-using-remix-and-metamask-image-10.png)

The contract ABI and Bytecode are available on the compiler tab.

![](./assets/avalanche-deploy-a-smart-contract-on-avalanche-using-remix-and-metamask-image-11.png)
